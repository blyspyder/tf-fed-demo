# tf-fed-demo
A federated learning demo for AlexNet on CIFAR-10 dataset, basing on Tensorflow.

## Dependence
1. Python 3.7
2. Tensorflow v1.14.x
3. tqdm

## Usage
```bash
cd ./src
python Server.py
```
